# 作业

1. **JDBC编程的六个步骤是什么**

    1. 注册驱动
        `Class.forName("oracle.jdbc.driver.OracleDriver");`
    2. 获得连接
        `Connection conn = DriverManager.getConnection(url, user, password);`
    3. 获得语句执行工具
        `Statement stmt = connection.createStatement();`
    4. 执行sql语句
        `ResultSet rs = stmt.executeQuery(sql);`
    5. 处理结果集(select)
        `while(rs.next()) {...}`
    6. 释放资源
        `rs.close();stmt.close();conn.close();`

2. **三种执行方法的返回值分别是什么**

    1. `statement.execute()` 返回boolen值
    2. `statement.executeUpdate()` 返回int值
    3. `statement.executeQuery()` 返回ResultSet

3. **请简述一下 Statement 和 PreparedStatement 之间的关系和区别**

    1. 关系：PreparedStatement继承自Statement,都是接口
    2. 区别：PreparedStatement可以使用占位符，是预编译的，批处理比Statement效率高

4. **描述一下 Driver，Drivermanager，Connection，Statement，ResultSet 含义**

    1. Driver接口
    2. DriverManager（管理一组 JDBC 驱动程序的基本服务）
    3. Connection（与特定数据库的连接。在连接上下文中执行 SQL 语句并返回结果）
    4. Statement（用于执行静态 SQL 语句并返回它所生成结果的对象。）
    5. ResultSet（表示数据库结果集的数据表，通常通过执行查询数据库的语句生成）
